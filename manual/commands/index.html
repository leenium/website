Prefix model

  • leenium-launch-*: open/focus applications and desktop tools
  • leenium-update*: update flows and analysis helpers
  • leenium-refresh-*: re-apply specific default config surfaces
  • leenium-restart-*: restart single runtime components
  • leenium-theme-* and leenium-font-*: appearance controls
  • leenium-hw-*: hardware detection and hardware-specific behavior helpers

Recovery-focused commands

  • leenium-restart-waybar, leenium-restart-walker, leenium-restart-mako
  • leenium-refresh-pacman, leenium-refresh-limine, leenium-refresh-plymouth
  • leenium-first-run when first-login automation did not complete
  • leenium-reinstall-configs for config drift recovery (broad, use carefully)
  • leenium-reinstall for payload-level recovery after logs confirm broad inconsistency
