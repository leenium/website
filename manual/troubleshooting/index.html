Logs first, targeted fixes second.

Use the smallest recovery command that matches the failure mode. Save broad resets for last.

Recovery order

  1. Read the relevant log file
  2. Run the smallest targeted fix
  3. Collect debug info before broad reset commands
  4. Escalate to reinstall-style commands only when needed
