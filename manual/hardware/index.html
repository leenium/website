Hibernation and resume

  • Check support with leenium-hibernation-available
  • Enable with leenium-hibernation-setup
  • Remove with leenium-hibernation-remove
  • If resume fails, collect leenium-debug --print and upload logs before changing swap or kernel parameters

Known-behavior guidance

  • Hybrid GPU laptops may need explicit app launch strategy; test using the default launch commands first
  • External monitor state changes are often transient; rerun monitor helper commands before broad config resets
  • Touchpad behavior can vary by vendor firmware; check leenium-hw-touchpad and related input overrides
  • For model-specific issues, include hardware detection command outputs when reporting
