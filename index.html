gpg --verify leenium.iso.sig leenium.iso

Choose your install path.

The ISO is the cleanest route, but it is not the only one. Leenium still supports a live-Arch path and an existing-Arch path for people who already have a base system.

Full disk install

Method 1 — Boot the Leenium ISO

Download the image, verify it, flash it to USB, and boot directly into the Leenium installer. This is the straight path for a fresh machine.

dd if=leenium.iso of=/dev/sdX \ bs=4M status=progress conv=fsync
Full disk install

Method 2 — Start from the official Arch ISO

If you prefer the stock Arch live environment, boot it, clone the repo, and run the installer from there.

git clone https://github.com/leenium/leenium cd leenium chmod +x install.sh ./install.sh
Keeps partitions

Method 3 — Layer it onto existing Arch

Already have Arch installed? Keep your base system and run Leenium on top. Recommended baseline: btrfs, limine, no desktop profile.

archinstall git clone https://github.com/leenium/leenium cd leenium ./install.sh

The repos behind the distro.

Leenium is split by responsibility: desktop config, ISO builder, package builder, package publishing, and the website itself.

leenium/leenium Main desktop repo: installer, scripts, Hyprland config, shipped environment. leenium/iso Builds the bootable ISO and stages the package flow used by the installer. Leenium GitHub org Entry point for the rest of the project repos and release-facing work. Hyprland config docs Reference for how the desktop is structured after install.

How Leenium ships updates.

The distro is split into clear operational surfaces so build, sign, publish, and install responsibilities stay isolated and debuggable.

1. Build + sign packages

packages-builder builds packages, signs every output, and manages the Leenium repo key workflow.

2. Publish repo metadata

pkg-repo ingests signed artifacts and publishes to https://pkgs.leenium.org/stable/x86_64/.

3. Build the ISO

iso consumes the live package repo and bootstraps leenium-keyring before package resolution.
