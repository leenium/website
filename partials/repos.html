leenium/leenium
github.com/leenium/leenium

The main repo. Install pipeline, bin/ commands, Hyprland config, themes, desktop entries — everything that runs on your machine. Clone it, run install.sh on a fresh Arch base.

leenium/iso
github.com/leenium/iso

Builds the bootable ISO with Docker + ArchISO. make build to build, make boot to test in a VM. The ISO does disk setup and kicks off the leenium install without any manual steps.

leenium/leenium.nvim
github.com/leenium/leenium.nvim

Neovim colorscheme. Deep navy-black background with teal-green accents. Install with any plugin manager: "leenium/leenium.nvim".
