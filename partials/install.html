Install paths

Pick the route that matches your current machine state: full ISO path, Arch live path, or existing Arch layering.

Download ISO Checksum Signature 
sha256sum -c leenium.iso.sha256sum
gpg --verify leenium.iso.sig leenium.iso

Method 1 - Leenium ISO

Full-disk install with the shortest path from USB boot to first login.

dd if=leenium.iso of=/dev/sdX bs=4M status=progress conv=fsync

Method 2 - Arch live ISO

Start from official Arch image then run Leenium installer.

git clone https://github.com/leenium/leenium
cd leenium
./install.sh

Method 3 - Existing Arch

Preserves partition layout and layers Leenium on top of your current Arch base.

archinstall
git clone https://github.com/leenium/leenium
cd leenium
./install.sh
