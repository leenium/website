Download the current x86_64 ISO

Leenium is currently published as a single x86_64 image. Download the ISO directly from the CDN, then verify its checksum and signature before writing it to USB media.

Download ISO SHA256 File Signature File
Artifacts live on cdn.leenium.org and include the ISO, detached signature, and SHA256 checksum file.
Verification

Verify the checksum file:

sha256sum -c leenium.iso.sha256sum

Then verify the detached signature:

gpg --verify leenium.iso.sig leenium.iso
