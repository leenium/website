user@leenium:~$ about

Started as a fork of Omarchy (DHH's setup), went its own direction pretty fast. Not a distro, not a rice — just a scripted install that sets up Arch + Hyprland exactly how I want it. Installs what it ships, configures what it touches, stays out of your way after.

How it works

Staged pipeline: preflight checks, then packages, then dotfiles and config, then login/display setup, then post-install. Source is open — you can read every step before running it.

Hardware

Has workarounds for Framework 16, ASUS ROG, Dell XPS OLED, Apple displays, Intel Panther Lake. Detects what you're on and applies the right fixes without asking.

Philosophy

Super+Enter opens a terminal. Super+K shows keybindings. You shouldn't need a mouse. If you do, there's probably a binding you haven't learned yet.

Keeping it updated

leenium-update handles packages. Config migrations run between versions. If things drift, leenium-reinstall resets the baseline.
