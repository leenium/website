Hyprland layering

Edit user overrides, not project-owned defaults. This keeps updates and local tweaks from fighting each other.

Load order

  • ~/.local/share/leenium/default/hypr (baseline)
  • ~/.config/leenium/current/theme/hyprland.conf (theme layer)
  • ~/.config/hypr/*.conf (user overrides)

Useful commands

hyprctl reload
leenium-menu-keybindings --print
leenium-reinstall-configs
Use reinstall commands intentionally; they can reset config state.
