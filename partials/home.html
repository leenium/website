I got tired of setting up Hyprland from scratch every time, so I wrote this. Run it after archinstall and you get a full desktop — packages, dotfiles, Waybar, themes, all of it — without spending a weekend on config files.

Hyprland, properly set up

Workspaces 1–10, scratchpad, tiling groups, floating, pin. Gaps and borders actually configured. Super+Enter opens a terminal, not a browser.

Mouse is optional

Browser, files, editor, music, messaging, passwords, AI — all bound. If you're reaching for the mouse you probably just forgot the shortcut.

Theming that actually works

leenium-theme-set, leenium-font-set, leenium-bg-set. One command and Waybar, terminal, Hyprland, lock screen all update together.

leenium-* commands

Update, screenshot, screen record, restart components, toggle nightlight, share — all just leenium-* in PATH. No hunting for scripts.

Install pipeline

preflight → packaging → config → login → post-install
